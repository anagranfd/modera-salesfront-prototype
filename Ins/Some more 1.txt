Что это за дизайн-система
Это не готовая публичная дизайн-система. Это кастомная тема поверх jQuery UI + Bootstrap 2.x, сделанная специально для Modera, скорее всего эстонской командой разработки (судя по эстонскому языку в интерфейсе). Приложение выглядит как ~2013–2015 год.
Конкретные маркеры, которые я вижу:

Синие градиентные хедеры панелей с белым текстом — классический jQuery UI dialog с кастомным скином
Кнопки с закруглёнными углами, градиентом и зелёной галочкой — Bootstrap 2.x с кастомизацией
Таббар с активным табом на синем фоне — Bootstrap 2 nav-tabs переокрашенные
Чекбоксы и радиокнопки — нативные, без кастомизации
Сетка таблицы — похожа на DataTables с кастомным CSS
Общий стиль с тенями и градиентами — типично для 2012–2014, до flat-дизайна


Что тебе реально поможет для тестового задания
Не ищи точный кит — его нет в публичном доступе. Вместо этого вот что реально подойдёт:
Вариант 1 — лучший: Figma Community > "Admin Dashboard UI Kit" от Untitled UI или Ant Design в стиле enterprise. Цвета подгони под Modera вручную (#263344 тёмный бар, #3a8fad голубой).
Вариант 2: Поищи в Figma Community по запросу "Enterprise CRM UI Kit" или "B2B SaaS Dashboard" — там есть компоненты с нужной плотностью информации.
Вариант 3 — самый прагматичный: Ты уже имеешь готовый HTML с точными цветами и компонентами, который мы с тобой построили. Это и есть твоя design reference. Просто перенеси токены оттуда в Figma вручную — займёт 20 минут.
Мой совет: не трать время на поиск "того самого кита". Судьи в Modera оценят точность воспроизведения их собственного стиля, а не соответствие чужому UI-киту. Используй наш HTML как живую референс-точку.

Что это за стек
Modera Salesfront построен на jQuery UI с кастомной темой через ThemeRoller + иконки Fugue от Yusuke Kamiyamane. Вот разбивка по элементам:

Градиентные элементы > jQuery UI ThemeRoller
Все эти синие хедеры панелей/диалогов с градиентом, закруглённые кнопки, табы с активным состоянием — это классический jQuery UI с кастомной ThemeRoller-темой. Конкретно стиль близок к теме "Cupertino" или "Flick" из галереи ThemeRoller, но перекрашенной в корпоративный сине-стальной цвет Modera.
Узнаваемые маркеры jQuery UI ThemeRoller:

Синий градиентный хедер диалога (.ui-widget-header)
Кнопка Close с крестиком ? в углу модала
Закруглённые кнопки с hover-состоянием
Характерный спрайт иконок ui-icons_*.png


Маленькие иконки > Fugue Icons (Yusuke Kamiyamane)
Fugue Iconpack — набор из 3570 иконок 16?16px от дизайнера Yusuke Kamiyamane IconArchive, лицензия CC Attribution. Это был один из самых популярных бесплатных наборов для enterprise web-приложений в 2010–2014 годах. Именно эти маленькие цветные 16px иконки — машинки, галочки, папки, стрелки — ты видишь по всему интерфейсу Modera.

Что тебе реально поможет для тестового
Для иконок — Fugue Icons до сих пор доступны бесплатно: iconarchive.com > поиск "fugue". Но для Figma-дизайна они не нужны — ты делаешь UX-спецификацию, а не pixel-perfect копию.
Для стиля в Figma — не ищи jQuery UI кит. Вместо него что-нибудь подобное из доступного сегодня.
