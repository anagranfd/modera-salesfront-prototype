Как организовать файл в Figma

  1. Создай три Section на канвасе — по одному на каждый эпизод. Расположи их вертикально сверху вниз: EP1 → EP2 → EP3.
  2. Внутри каждого Section расставь фреймы по рядам (слева направо = ход сценария). Оставляй между фреймами 80–100px по горизонтали, 120px между рядами.
  3. Overlay-экраны (Channel Select, Unpublish Select/Confirm) размещай над или чуть правее родительского фрейма, из которого они открываются.
  4. Соединяй фреймы стрелками Figma (R → Connections). На каждой стрелке добавь текстовую метку — название действия пользователя.
  5. К каждому фрейму прикрепи Sticky Note (или текстовый блок снизу) с четырьмя строками: Entry / Показывает / Действия / → Переходы. Цветовую схему стикеров смотри в легенде ниже.
  6. Назови каждый фрейм точно по шаблону: EP1 · A1 · Idle (Unpublished) — код, точка, имя.
